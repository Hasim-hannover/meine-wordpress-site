# Workflows

Hier liegen exportierte n8n-JSONs.

Benennungsregel:

- `<domain>__<purpose>__<status>.json`

Beispiele:

- `audit-funnel__start-analysis__live.json`
- `audit-funnel__status-polling__mvp.json`
- `lead-routing__crm-handoff__refactor.json`
- `audit-funnel__instant-results__target.json`

Wichtig:

- JSON allein ist nie genug.
- Jede Datei braucht eine gleichnamige Doku unter `../docs/`.
