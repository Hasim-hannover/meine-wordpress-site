# Blog Drafts

Hier liegen Rohfassungen fuer Blog- und Cornerstone-Inhalte.

Regel pro Draft:

- klare Zielgruppe
- klares Such- oder Entscheidungsproblem
- primaerer CTA
- Bezug zu Service oder Funnel-Stufe
- Status: Entwurf, im Editor, live oder verworfen

Empfehlung:

- Markdown bevorzugen, wenn der Entwurf noch redaktionell ist.
- HTML nur dann ablegen, wenn bereits ein strukturiertes Publish-Paket vorliegt.
