# Content

Hier liegt Content, der noch nicht sauber als Live-Content im WordPress-Editor oder als finale Repo-Doku eingeordnet ist.

Aktuelle Rolle:

- Rohfassungen

Wichtige Grenze:

- Der WordPress-Editor ist ein externer Content-Layer.
- Wenn Live-Content von Repo-Drafts abweicht, muss das bewusst dokumentiert werden.
